# Setup Instructions
Things you may want to cover:
 
 * Node Version
   v14.16.1
 
 * System dependencies
   MongoDB
 
 * Configuration
   Make sure to point MONGO_CONNECTION_STRING environment to valid MongoDB connection
   ```
    npm install
   ```
   
# How to run the test suite
  ```
  npm test
  ```
  
# Loading Seed
  ```
  node -r esm seed/userWithHobbies.seed.js
  ```

# Run Dev App
  ```
  npm run devStart
  ```
  
# Run Production App
  ```
  npm start
  ```
